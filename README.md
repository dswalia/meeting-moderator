# Meeting Moderator

**Meeting Moderator** is a tool designed to streamline meetings by providing real-time transcription and speech recognition. This tool can assist development teams stay on track, document discussions, and improve meeting efficiency.

Current focus is limited to the use-case of Standup meetings, which would later be extended to cover other variations of meeting.

## Features

- **Automated Timekeeping:** Keep your Scrum meetings on schedule with built-in timers.
- **Real-Time Transcription:** Transcribe team discussions as they happen.
- **Speech Recognition:** Convert spoken words to text for accurate meeting notes.
- **Meeting Summaries:** Generate summaries and action items automatically.
